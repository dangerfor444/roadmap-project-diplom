Демо встраивания виджета

Эта страница показывает, как передавать identity из сайта-хоста в iframe через embed.js и postMessage.

Snippet (prod-like)

<script
  src="http://localhost:1337/embed.js"
  data-project="default"
  data-target="#widget-host"
  data-height="760"
  data-frame-id="roadmap-widget-frame"
  data-api-name="RoadmapEmbed"
  data-actor-token="<signed-token>"
></script>

<script>
  // optional runtime refresh (for token rotation)
  window.RoadmapEmbed.setAuth({ actorToken: '<new-token>' });
</script>

Живой пример

Кнопка вызывает window.RoadmapEmbedDemo.setAuth(...) без перезагрузки виджета.
